# RTC 电池

瑞莎 X4 支持外接 RTC（实时时钟）纽扣电池，具体接口请参见产品标注图。

## 接口座子规格

- RTC 电池连接座：2-Pin，间距为 1.25mm

## 电池规格

![](/img/accessories/rtc_battery.webp)

- RTC 电池型号：CR2032
- RTC 电池电压：3V
- RTC 电池连接器型号：MX1.25mm 公头

:::tip
请注意，RTC 连接座不支持充电功能。
:::
