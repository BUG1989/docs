- RS485 USB 串口接到 PC 端， {props.product} 设置 RS485 波特率为 115200

- PC端打开串口工具，并设置好对应的串口和波特率，这里以 Linux 为例

- {props.product} 发送任意值，主机端串口打印均可以收到
