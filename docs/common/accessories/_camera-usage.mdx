- 连接

  通过 FPC 线接上 {props.product} ，如图所示：

  <img
    src={props.camera_connection_img}
    alt="camera connection"
    style={{ width: "80%" }}
  />

- 开启 Overlay

  1. 通过左下角 Application Launcher 打开 Kconsole 终端, 运行 rsetup 命令：

  <pre> radxa@{props.model}:~$ rsetup </pre>

  2. 通过 <a href={props.rsetup_path}> 设备树配置 </a>来启用 {props.camera} 的 Overlay。

  :::tip

  请启用 [] {props.overlays_title} 项 Overlay。<br/>
  在启用成功显示 [*] {props.overlays_title} 后退出重启才能使配置生效。

  :::

  3. 重启系统

- 测试

  1. 预览

  打开终端，输入以下命令打开相机预览:

  <pre>
    gst-launch-1.0 v4l2src device={props.video_dev} io-mode=4 ! videoconvert !
    video/x-raw,format=NV12,width=1920,height=1080 ! xvimagesink;
  </pre>

  2. 拍照

  打开终端，输入以下命令打开相机进行拍照:

  <pre>
    gst-launch-1.0 v4l2src device={props.video_dev} io-mode=4 ! videoconvert !
    video/x-raw,format=NV12,width=1920,height=1080 ! jpegenc ! multifilesink
    location=file.name.jpg;
  </pre>

  3. 拍摄视频

  打开终端，输入以下命令打开相机进行拍摄视频:

  <pre>
    gst-launch-1.0 v4l2src num-buffers=512 device={props.video_dev} io-mode=4 !
    videoconvert ! video/x-raw, format=NV12, width=1920, height=1080,
    framerate=30/1 ! tee name=t ! queue ! mpph264enc ! queue ! h264parse !
    mpegtsmux ! filesink location=/home/radxa/file.name.mp4
  </pre>
