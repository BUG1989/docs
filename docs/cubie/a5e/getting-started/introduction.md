# 产品介绍

CUBIE A5E 是瑞莎推出的一款紧凑型单板计算机（SBC），搭载全志 A527 或 T527 处理器，采用 8 核 Arm Cortex-A55 CPU，并配备 Arm Mali-G57 MC1 GPU。 T527 版本还集成了 2 TOPS（INT8）算力的 NPU 和 Cadence Tensilica HiFi4 DSP，适合 AI 应用和工业场景。
