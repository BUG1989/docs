# ROCK 5 ITX

## 文档使用指引

:::tip
除了下面的 **其他系统** 目录外，其他内容都是围绕 Radxa OS (Radxa 基于 Debian 定制的系统) 展开。
:::

介绍系统烧录，系统登录，配件使用等内容让用户能快速的上手使用。

方便用户快速的熟悉并使用系统。

### [应用开发](./app-development/README)

介绍上层应用开发，比如QT, WiringX, Mraa 等。

### [硬件开发](./hardware-design/README)
