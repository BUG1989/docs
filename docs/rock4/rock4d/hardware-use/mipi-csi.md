# MIPI CSI

Radxa ROCK 4D 板载 两个MIPI CSI 接口，我们可以通过 MIPI CSI 接口连接摄像头。

其中两个 MIPI CSI 接口规格分别是 4-lane(4 通道)MIPI CSI 和 2-lane(2 通道) MIPI CSI，4-lane MIPI CSI 接口支持更高分辨率的摄像头模块，用户可以根据实际需求选择使用。
