# Power Selection

ROCK 2A supports various power supplies with a fixed voltage input of 5V:

- Radxa PD30W

- Apple Adapter 20W

- Anker Adapter 65W

- Xiaomi Adapter 65W

- Ugreen Adapter 65W

- Baseus Adapter 65W

:::tip
It is recommended to use the official 30W adapter [Radxa PD 30W Adapter](../../../accessories/pd_30w)
:::
