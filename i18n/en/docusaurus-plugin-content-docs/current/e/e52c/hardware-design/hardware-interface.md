| 4   | VCC_3V3    |
| 5   | SDMMC_CLK  |
| 6   | GND        |

| 9   | SDMMC_DETN |

| 11  | GND        |

| 13  | GND        |

| PIN | Name            |

| 3   | TYPEC0_OTG_DP   |

| 5   | TYPEC0_SSRX1N   |
| 6   | TYPEC0_SSRX1P   |
| 7   | GND             |
| 8   | TYPEC0_SSTX1N   |
| 9   | TYPEC0_SSTX1P   |
