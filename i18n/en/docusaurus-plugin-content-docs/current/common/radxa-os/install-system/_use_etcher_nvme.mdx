This guide explains how to install the system onto an NVMe SSD using an M.2 NVMe SSD enclosure.

:::warning
Installing the system will format the NVMe SSD. Please back up any important data in advance!
:::

## Prerequisites

### Hardware Requirements

You will need the following hardware devices:

- M.2 NVMe SSD enclosure
- Bootable storage medium: M.2 NVMe SSD

### Hardware Connection

Install the M.2 NVMe SSD into the M.2 NVMe SSD enclosure, then connect the enclosure to your computer.

## System Installation

### Download System Image

Use a PC to access the [Resource Downloads](../../../download) page and download the system image file locally.

:::tip Image Note
The downloaded system image is a compressed file and needs to be extracted before use!
:::

### Install System Image

You can use Balena Etcher software to install the system image onto the M.2 NVMe SSD.

:::tip Balena Etcher
For Balena Etcher installation and usage instructions, please refer to the [Etcher Guide](/common/radxa-os/install-system/balena-etcher) tutorial.
:::
