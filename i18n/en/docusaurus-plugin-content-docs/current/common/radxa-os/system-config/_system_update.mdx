We use the `Rsetup` tool to update the system.

:::tip
Using the `Rsetup` tool provides a safer way to update your system.

Using the `sudo apt update && sudo apt upgrade` command to upgrade the system may result in incomplete updates or system instability.

For more information about using the `Rsetup` tool, refer to:

- [Rsetup Tool](./rsetup)
  :::

## Rsetup Tool

Open a terminal and enter the following command to launch the `Rsetup` tool:

<NewCodeBlock tip="radxa@device$" type="device">

```
sudo rsetup
```

</NewCodeBlock>

## Updating the System

After launching the `Rsetup` tool, navigate to `System` -> `System Update` and follow the on-screen prompts to complete the system update process.
