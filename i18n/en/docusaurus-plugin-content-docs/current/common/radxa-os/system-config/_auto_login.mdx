This document explains how to set up automatic desktop login using the `Rsetup` tool on Radxa boards.

:::tip
Enabling automatic desktop login allows the system to log in automatically without requiring a username and password on each boot.

For more information about the `Rsetup` tool, please refer to:

- [Rsetup Tool](./rsetup)
  :::

## Rsetup Tool

Open a terminal and enter the `sudo rsetup` command to launch the `Rsetup` tool:

<NewCodeBlock tip="radxa@device$" type="device">

```
sudo rsetup
```

</NewCodeBlock>

## Configure Auto Login

After launching the `Rsetup` tool, navigate to `User Settings` -> `Configure auto login`, enable `sddm.service`, and follow the on-screen instructions to complete the setup.
